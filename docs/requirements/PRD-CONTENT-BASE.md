# 3. 언어스킬 도출

## 1. 전제

- 시작수준 : 3세 정도의 단어와 언어구사 수준을 시작수준으로 함
- 목표수준 : 대학교 학생 수준의 원어민 언어실력
- 타겟 : 제2외국어로 다른 언어를 배우며, 하루에 평균적으로 30~60분 정도 시간을 투여할 수 있음.

## 2. 언어스킬 도출 및 정리

- 시작수준부터 목표수준까지 도달하기 위해 다양한 주요언어에서 공통적으로 익혀야할 스킬을 최대한 자세히 분류한 후 해당 스킬이 300~500여개 정도가 되도록 구조적이고 순차적으로 정리해야 함.
- 단어학습 : 기초레벨에서 문장을 구성하기 위한 최소 단어는 기초스킬 (1 ~ 10단계)까지 문장으로 취급하여 학습함. 이후 10단계 이후가 되면 새로운 단어는 힌트로 설명함.

## 3. 스킬유형 정리

- 형태 (단어, 문장), 시제 (현재, 과거, 과거완료 등), 목적/용도 (질문, 권고, 명령, 설득, 설명, 요청 등), 장소/상황 (집, 회사, 교육, 여행) 등 언어사용의 목적을 유형화할 수 있는 스킬유형을 대구분, 소구분으로 도출/정리함.
- 스킬 유형은 이후 각 학습자별 강약점을 레이다차트 형태로 분석해서, 부족한 점을 집중적으로 학습할 수 있도록 하기 위해 사용됨.
- 대구분 (시제, 용도, 상황 등)은 가능한 모든 스킬에 공통적으로 적용될 수 있는 항목이어야 함. 단, 기초스킬에 해당 하는 단어형태는 해당 안됨.
- 학습을 위한 유형이 스킬레벨에 있는 것이 맞을지 아니면 문장 레벨에 있는 것이 나을지 확인 (일부 유형은 스킬에벨에 두고 일부유형은 문장레벨에 두는 안도 검토)

## 4. 스킬레벨 정리

- 스킬을 1 ~ 100레벨까지 레벨별로 나누어 정리함. 이때 각 레벨별로 유사한 개수의 스킬이 배정되어야 함. (3~5개)
- 가장 기초적인 스킬을 1, 가장 높은 스킬을 100레벨에 배치함.

## 5. 스킬 점검

- 도출된 스킬이 각 언어별 학습을 고려할 때 빠지거나 중복된 것이 없는지 각 언어학자 에이전트를 이용하여 점검하고 수정이 필요한 부분은 수정함.
- 이때 개별 언어별로 미세한 차이는 허용하나 2개 이상의 언어에서 사용하지 않는 중요성이 떨어지는 스킬은 제외함.

## 6. 스킬 최종정리

- 스킬은 다음과 같은 정보로 구성함
  - ID : 스킬순번 (1번부터 순번), 식별자
  - 레벨 : 1 ~ 100까지 정수
  - 스킬명 : 한글 30글자
  - 스킬개요 : 스킬에 대한 구체적인 설명으로 200글자 이내
  - 스킬유형 : 대구분별 소구분값 설정

# 4. 문장 도출

## 1 적절한 문장개수 정의

- 각 스킬을 학습하기에 적절한 스킬별 문장 개수를 스킬별로 산정함. (최소 5개 이상)

## 2 기본문장 도출

- 정리된 스킬별로 스킬을 학습하기 위한 최적의 문장을 스킬별 문장수에 맞추어 도출함.
- 문장은 2개 이상의 단어를 의미하나 기초스킬레벨에서는 단어를 학습하기도 하기 때문에 단어 1개도 문장으로 취급함.
- 해당 문장은 이전에 학습한 내용(단어, 문장)을 적절힌 반복사용하도록 내용을 구성해서 점진적 학습효과를 극대화하도록 내용을 구성해야 함.

## 3 문장 점검

- 도출된 문장이 각 스킬별 학습을 고려할 때 모자라는 부분이 없을지, 충분히 스킬학습을 보장하는지 각 언어학자 에이전트를 이용하여 점검하고 수정이 필요한 부분은 수정함.

## 4 응용문장 도출

- 응용문장 : 문장 내 한 두개 단어만 변경하여 학습노력은 줄이면서도 암기와 내재화에는 도움이 되도록 하는 문장을 응용문장이라고 함.
- 도출된 문장별로 한 두개의 단어만 변형한 응용문장을 문장당 3개 추가로 생성함. 이렇게 생성된 문장은 응용문장유형이며, 기본문장ID 값을 저장하고 있어서 추후 학습에 활용함.

## 5. 문장최종정리

- 문장은 다음과 같은 정보로 구성됨
  - ID : 식별자
  - 스킬ID : 스킬ID
  - 문장형태 : 기본문장, 응용문장
  - 기본문장ID : 문장ID
  - 의미 : 한글로된 문장

# 5. 언어별 표현 정리

## 1. 언어별 표현정리

- 영어, 스페인어, 일어, 중국어, 한국어 5개국어로 각 문장에 맞는 영어표현 작성
- 이때 문장이 포함된 레벨, 스킬 수준에 맞는 단어와 표현 사용
- 힌트 : 문장 내 새로나온 어려운 단어나 숙어 같은 경우 설명하는 내용으로 100글자 이내 설명

## 2. 표현 점검

- 도출된 표현이 각 스킬별 학습 및 문장을 고려할 때 적합한지 최신 언어상황과 실생활 실용언어 관점에서 적절한지 언어학자 에이전트를 이용하여 점검하고 수정이 필요한 부분은 수정함.

## 3. 표현 최종정리

- 표현은 다음과 같은 정보로 구성됨
  - ID : 식별자
  - 문장ID : 문장ID
  - 언어 : 언어유형ID
  - 힌트 : 200글자 이내
