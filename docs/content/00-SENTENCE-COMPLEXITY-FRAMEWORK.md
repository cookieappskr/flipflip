# FlipFlip 문장 복잡도 진행 프레임워크 (Sentence Complexity Progression Framework)

> **작성일**: 2026-03-15
> **버전**: v1.0
> **목적**: L1부터 L100까지 문장 복잡도가 급격한 단절 없이 부드럽게 상승하도록 하는 기준 문서
> **대상**: 콘텐츠 작성자 (Content Writers), 언어학자 에이전트 (Linguist Agents)
> **핵심 원칙**: 학습자가 어떤 레벨에서도 "갑자기 어려워졌다"고 느끼지 않도록 점진적 곡선(Gradual Curve)을 유지한다.

---

## 1. 복잡도 티어 정의 (Complexity Tier Definitions)

| 티어 | 명칭 (KR) | 명칭 (EN) | 정의 | 예시 | 평균 음절/글자 수 |
|------|-----------|-----------|------|------|-------------------|
| **T1** | 단어 | Single Word | 단일 어절 또는 고정 인사 표현 | 물, 안녕하세요, 네 | 1-5음절 |
| **T2** | 짧은 구 | Short Phrase | 2-3어절의 조합. 조사 또는 관형사 포함 가능. 완전한 서술어 없음 | 좋은 아침, 큰 집, 물 주세요, 이쪽으로요 | 3-8음절 |
| **T3** | 단문 | Simple Sentence | 주어+서술어를 갖춘 1절(clause) 문장. 목적어/보어 1개 이하 | 나는 학생입니다, 비가 옵니다 | 5-12음절 |
| **T4** | 확장 단문 | Extended Simple Sentence | 1절 문장이나 부사어, 시간/장소 표현 등으로 확장 | 나는 매일 아침 학교에 갑니다 | 8-18음절 |
| **T5** | 중문 | Compound Sentence | 2개의 독립절이 등위접속사(-고, -지만, 그리고, 하지만)로 연결 | 나는 커피를 좋아하지만 차도 마십니다 | 12-25음절 |
| **T6** | 복문 | Complex Sentence | 종속절 + 주절 구조. 조건(-면), 이유(-서, -니까), 양보(-지만), 시간(-때) 등 | 비가 오면 집에서 책을 읽을 것입니다 | 15-30음절 |
| **T7** | 다중절 | Multi-clause Sentence | 3개 이상의 절. 고급 연결어미(-건대, -을뿐더러, -거니와), 인용, 간접화법 포함 | 자원이 부족한바, 우선순위를 재조정하되 핵심 사업은 유지해야 합니다 | 20-45음절 |

## 2. 레벨별 티어 배분표 (Level-by-Level Tier Distribution)

### 2.1 배분 원칙

1. **인접 레벨 간 변동폭**: 어떤 티어의 비율이 인접 레벨 범위에서 30%p 이상 급변하지 않도록 한다.
2. **신규 티어 도입**: 새로운 티어는 먼저 10-20%로 "맛보기" 도입한 후, 다음 레벨 범위에서 비율을 높인다.
3. **퇴장 티어**: 이전 티어는 즉시 0%로 떨어지지 않고 점진적으로 감소한다.

### 2.2 배분표

| 레벨 범위 | T1 (단어) | T2 (짧은 구) | T3 (단문) | T4 (확장 단문) | T5 (중문) | T6 (복문) | T7 (다중절) | 주요 특징 |
|-----------|-----------|-------------|-----------|---------------|-----------|-----------|-------------|-----------|
| **L1-L3** | **100%** | - | - | - | - | - | - | 기초 단어, 인사, 숫자 |
| **L4-L5** | **60%** | **40%** | - | - | - | - | - | 짧은 구 첫 도입 |
| **L6-L7** | **30%** | **40%** | **30%** | - | - | - | - | 첫 단문 도입 (주어+서술어) |
| **L8-L10** | **15%** | **30%** | **45%** | **10%** | - | - | - | 단문 주력, 확장 단문 맛보기 |
| **L11-L15** | - | **10%** | **40%** | **40%** | **10%** | - | - | 확장 단문 주력, 중문 첫 도입 |
| **L16-L20** | - | **5%** | **20%** | **35%** | **30%** | **10%** | - | 중문 본격화, 복문 첫 도입 |
| **L21-L25** | - | - | **10%** | **25%** | **35%** | **25%** | **5%** | 복문 확대, 다중절 맛보기 |
| **L26-L30** | - | - | **5%** | **15%** | **35%** | **35%** | **10%** | 중문+복문 균형 |
| **L31-L40** | - | - | - | **10%** | **30%** | **40%** | **20%** | 복문 주력 |
| **L41-L50** | - | - | - | **5%** | **20%** | **40%** | **35%** | 다중절 본격화 |
| **L51-L60** | - | - | - | - | **15%** | **35%** | **50%** | 다중절 과반 |
| **L61-L70** | - | - | - | - | **10%** | **30%** | **60%** | 고급 구조 중심 |
| **L71-L80** | - | - | - | - | **5%** | **25%** | **70%** | 학술/비즈니스 수준 |
| **L81-L90** | - | - | - | - | - | **20%** | **80%** | 전문적 복합 구조 |
| **L91-L100** | - | - | - | - | - | **10%** | **90%** | 원어민 근접, 고급 연결어미 |

## 3. 연결어/패턴 도입 로드맵 (Connector/Pattern Introduction Roadmap)

### 3.1 등위 접속 (Coordinating Connectors)

| 도입 레벨 | 연결어/어미 | 기능 | 예시 |
|-----------|-------------|------|------|
| L8 | -고 | 나열 (and) | 밥을 먹고 학교에 갑니다 |
| L11 | 그리고 | 나열 (and, sentence-initial) | 그리고 커피도 주세요 |
| L11 | -지만 | 대조 (but) | 비싸지만 맛있습니다 |
| L13 | 하지만 | 대조 (but, sentence-initial) | 하지만 시간이 없습니다 |
| L16 | 또는 / -거나 | 선택 (or) | 커피 또는 차를 드릴까요? |
| L21 | -도 ...도 | 병렬 강조 (both...and) | 영어도 잘하고 한국어도 잘합니다 |

### 3.2 종속 접속 (Subordinating Connectors)

| 도입 레벨 | 연결어미/표현 | 기능 | 예시 |
|-----------|-------------|------|------|
| L16 | -서 / -아서/어서 | 이유 (because - casual) | 배고파서 밥을 먹었습니다 |
| L18 | -니까 / -으니까 | 이유 (because - emphatic) | 비가 오니까 우산을 가져가세요 |
| L20 | -면 / -으면 | 조건 (if) | 시간이 있으면 같이 가요 |
| L22 | -(으)ㄹ 때 | 시간 (when) | 집에 올 때 빵을 사 주세요 |
| L25 | -기 전에 / -은 후에 | 시간 (before/after) | 자기 전에 책을 읽습니다 |
| L28 | -기 때문에 | 이유 (because - formal) | 약속이 있기 때문에 먼저 가겠습니다 |
| L30 | -(으)면서 | 동시 (while) | 음악을 들으면서 공부합니다 |
| L33 | -도록 | 목적/정도 (so that) | 늦지 않도록 일찍 출발하세요 |
| L35 | -ㄹ지라도 / -더라도 | 양보 (even if) | 비가 오더라도 가겠습니다 |
| L38 | -는 반면에 | 대조 (whereas) | 형은 키가 큰 반면에 동생은 작습니다 |
| L40 | -(ㄴ/는)다고 / -라고 | 인용 (quotation) | 그가 내일 온다고 했습니다 |

### 3.3 고급 연결 표현 (Advanced Connectors)

| 도입 레벨 | 연결어미/표현 | 기능 | 예시 |
|-----------|-------------|------|------|
| L45 | -을 뿐만 아니라 | 추가 강조 (not only...but also) | 가격이 저렴할 뿐만 아니라 품질도 좋습니다 |
| L50 | -는 한 / -는 이상 | 조건 한정 (as long as) | 노력하는 한 성공할 수 있습니다 |
| L55 | -(으)ㄴ/는 셈이다 | 결론/추정 (it amounts to) | 매일 운동하니 건강한 셈이다 |
| L60 | -건대 / -거든 | 전제 (given that) | 솔직히 말하건대 결과가 좋지 않습니다 |
| L65 | -(으)ㄹ진대 | 양보 강조 (even granting) | 사정이 어려울진대 포기할 수는 없습니다 |
| L70 | -거니와 | 추가 (moreover, formal) | 시간도 부족하거니와 인력도 모자랍니다 |
| L75 | -을뿐더러 | 첨가 강조 (not only, literary) | 비용이 증가했을뿐더러 품질도 저하되었습니다 |
| L80 | -는바 / -ㄴ바 | 근거 제시 (as, given) | 수요가 급증한바 공급 체계를 확충해야 합니다 |
| L85 | -되 | 조건부 허용 (but/provided) | 참여하되 규칙을 준수해야 합니다 |
| L90 | 복합 연결 조합 | 3개 이상 연결어미 중첩 | (3.4 참고) |

### 3.4 연결어미 중첩 가이드 (Connector Stacking Guide)

| 레벨 범위 | 최대 연결어미 수/문장 | 설명 |
|-----------|---------------------|------|
| L1-L10 | 0-1 | -고 정도만 허용 (L8+) |
| L11-L20 | 1-2 | 기초 연결어미 1개, 후반부 2개 가능 |
| L21-L30 | 2 | 2개 연결어미 표준 |
| L31-L40 | 2-3 | 복문에서 2-3개 |
| L41-L60 | 3 | 다중절에서 3개 표준 |
| L61-L80 | 3-4 | 고급 연결어미 포함 3-4개 |
| L81-L100 | 4+ | 제한 없음, 자연스러운 범위 내 |

## 4. 문장 길이 가이드라인 (Sentence Length Guidelines)

### 4.1 레벨별 목표 길이

| 레벨 범위 | 평균 어절 수 | 최소 어절 | 최대 어절 | 평균 글자 수(공백 제외) | 참고 |
|-----------|-------------|-----------|-----------|----------------------|------|
| **L1-L3** | 1.0 | 1 | 1 | 2-5 | 단일 단어/표현 |
| **L4-L5** | 1.8 | 1 | 3 | 3-8 | 구와 단어 혼합 |
| **L6-L7** | 2.5 | 1 | 5 | 4-12 | 첫 단문 등장 |
| **L8-L10** | 3.5 | 1 | 7 | 6-18 | 단문 위주 |
| **L11-L15** | 5.0 | 3 | 9 | 10-25 | 확장 단문 + 초기 중문 |
| **L16-L20** | 6.5 | 4 | 11 | 14-30 | 중문 본격화 |
| **L21-L25** | 7.5 | 4 | 13 | 16-35 | 복문 확대 |
| **L26-L30** | 8.5 | 5 | 14 | 18-38 | 중문+복문 균형 |
| **L31-L40** | 9.5 | 5 | 16 | 20-42 | 복문 주력 |
| **L41-L50** | 11.0 | 6 | 18 | 22-45 | 다중절 본격화 |
| **L51-L60** | 12.5 | 7 | 20 | 25-50 | 다중절 과반 |
| **L61-L70** | 14.0 | 8 | 22 | 28-55 | 고급 구조 |
| **L71-L80** | 15.5 | 9 | 25 | 30-60 | 학술/비즈니스 수준 |
| **L81-L90** | 17.0 | 10 | 28 | 32-65 | 전문적 복합 구조 |
| **L91-L100** | 18.0 | 10 | 30 | 35-70 | 원어민 근접 |

### 4.2 길이 분포 규칙

- 같은 스킬 내 문장들의 어절 수 표준편차가 **평균의 40%를 초과하지 않도록** 한다.
- 예: L11-L15 평균 5어절이면, 한 스킬 내 문장들은 3-7어절 사이에 분포해야 한다.
- 레벨 범위 내에서 **짧은 문장과 긴 문장이 적절히 섞여야** 한다 (모두 최대 길이거나 모두 최소 길이이면 안 됨).

## 5. 레벨별 예시 문장 (Example Sentences by Level Range)

### L1-L3: 단어 (T1 100%)

```
L1: 안녕하세요 / 감사합니다 / 네 / 물 / 밥
L2: 하나 / 둘 / 아버지 / 어머니 / 친구
L3: 학교 / 집 / 사과 / 크다 / 작다
```

### L4-L5: 단어 + 짧은 구 (T1 60% + T2 40%)

```
[T1] 빨간색 / 파란색 / 위 / 아래
[T2] 좋은 아침 / 큰 집 / 물 주세요 / 이쪽으로요
[T2] 얼마예요? / 감사합니다, 선생님 / 안녕, 친구
```

### L6-L7: 단어 + 짧은 구 + 단문 (T1 30% + T2 40% + T3 30%)

```
[T1] 월요일 / 오늘 / 어디
[T2] 맛있는 음식 / 저 사람 / 여기 앉으세요
[T3] 나는 학생입니다 / 날씨가 좋습니다 / 이것은 책입니다
[T3] 물이 있습니다 / 그녀는 선생님입니다
```

### L8-L10: 구 + 단문 + 확장 단문 (T1 15% + T2 30% + T3 45% + T4 10%)

```
[T1] 내일 / 어제
[T2] 커피 한 잔 / 학교 앞에서 / 몇 시에요?
[T3] 나는 커피를 마십니다 / 그는 학생입니다 / 오늘 날씨가 좋습니다
[T3] 여동생이 있습니다 / 고양이가 좋습니다
[T4] 나는 매일 아침 학교에 갑니다 / 어머니는 시장에서 과일을 삽니다
```

### L11-L15: 단문 + 확장 단문 + 초기 중문 (T2 10% + T3 40% + T4 40% + T5 10%)

```
[T2] 조금만 기다려 주세요
[T3] 저는 서울에 삽니다 / 오늘 학교에 갑니다
[T4] 나는 매일 저녁 여섯 시에 집에 옵니다
[T4] 그녀는 아침마다 공원에서 운동합니다
[T5] 나는 피곤하지만 기분은 좋습니다
```

> **주의**: L11의 문장이 L10보다 급격히 길거나 복잡해서는 안 된다. L10의 확장 단문(T4)과 L11의 단문/확장 단문(T3/T4)이 자연스럽게 이어져야 한다.

### L16-L20: 확장 단문 + 중문 + 초기 복문 (T2 5% + T3 20% + T4 35% + T5 30% + T6 10%)

```
[T3] 비가 옵니다
[T4] 주말에 친구와 같이 영화를 봅니다
[T5] 나는 커피를 좋아하고 동생은 차를 좋아합니다
[T5] 공부는 어렵지만 재미있습니다
[T6] 배가 고파서 밥을 먹었습니다
```

### L21-L30: 중문 + 복문 중심 (자연스러운 일상 문장)

```
L21-L25:
[T4] 우리는 어제 피곤했습니다
[T5] 그 영화는 길었지만 재미있었습니다
[T6] 배가 고파서 점심을 일찍 먹었습니다
[T6] 지난주에 친구와 함께 영화를 봤습니다
[T7] 어제 오후에 친구를 만나서 커피를 마셨습니다 (2절 이하이므로 실제 T5-T6)

L26-L30:
[T5] 장을 보고 저녁을 요리했습니다
[T6] 시간이 없어서 아침을 먹지 않았습니다
[T6] 비가 오니까 우산을 가져가세요
[T7] 일찍 출발했기 때문에 우리는 늦지 않았습니다
```

> **핵심 원칙 (L20-L30)**: 이 구간의 문장은 **단순하고 자연스러운 일상 문장**이어야 한다. 교과서적으로 정돈된 짧은 문장이 주를 이루며, 인위적으로 복잡하게 만들지 않는다. "어제 친구를 만났습니다" 수준이 적절하며, "어제 친구를 만났는데 그 친구가 최근에 이사를 해서 새 집에서 같이 저녁을 먹었습니다" 수준은 L35+ 이상에서 사용한다.

### L31-L40: 복문 주력 + 다중절 도입

```
[T5] 일이 끝나고 저녁을 요리할 것입니다
[T6] 내일 비가 오면 집에서 영화를 볼 것입니다
```

## 6. 전환 구간 상세 가이드 (Transition Zone Details)

### 6.1 L3 → L4 전환 (단어에서 구로)

이 전환이 학습자가 처음으로 "조합"을 경험하는 지점이다.

| 전략 | 설명 | 예시 |
|------|------|------|
| 이미 학습한 단어 조합 | L1-L3에서 배운 단어를 결합 | 큰(L3) + 집(L3) → 큰 집 |
| 관형사 + 명사 | 가장 단순한 구 형태 | 좋은 아침, 맛있는 음식 |
| 명사 + 주세요/있어요 | 기초 요청/존재 표현 | 물 주세요, 빵 있어요? |

### 6.2 L5 → L6 전환 (구에서 단문으로)

| 전략 | 설명 | 예시 |
|------|------|------|
| 구에 서술어 추가 | 짧은 구에 -입니다/-있습니다 추가 | 큰 집 → 큰 집입니다 |
| 대명사 + 서술어 | 가장 짧은 완전한 문장 | 나는 학생입니다 |
| 주어 생략 가능 문장 | 한국어 특성 활용 | 좋습니다 / 맛있습니다 |

### 6.3 L10 → L11 전환 (핵심 전환 구간)

**이 구간은 가장 중요한 전환점이다.** L10의 마지막 문장과 L11의 첫 문장이 복잡도 면에서 연속적이어야 한다.

| L10 후반 예시 (T3/T4) | L11 초반 예시 (T3/T4) | 차이 |
|----------------------|---------------------|------|
| 나는 매일 학교에 갑니다 | 저는 서울에 삽니다 | 거의 동일한 복잡도 |
| 어머니는 시장에서 과일을 삽니다 | 그녀는 아침마다 공원에서 운동합니다 | 어절 수 1개 차이 |
| 오늘 날씨가 좋습니다 | 오늘 수업이 재미있습니다 | 동일한 구조 |

**금지 사항**: L11의 첫 스킬에 중문(T5)이나 복문(T6)을 넣지 않는다. L11 스킬의 첫 2-3개 문장은 반드시 T3 또는 T4여야 한다.

### 6.4 L30 → L31 전환 (중급 진입)

| L30 후반 예시 | L31 초반 예시 | 차이 |
|-------------|-------------|------|
| 시간이 없어서 택시를 탔습니다 | 내일 비가 오면 집에서 영화를 볼 것입니다 | 시제 변화 (미래), 구조 유사 |
| 비가 오니까 우산을 가져가세요 | 시간이 있으면 전화하겠습니다 | 조건절 구조 유사 |

## 7. 콘텐츠 작성 체크리스트 (Content Writer Checklist)

문장 작성 시 아래 항목을 반드시 확인한다.

### 7.1 개별 문장 검증

- [ ] 해당 레벨 범위의 **티어 배분 비율**에 맞는가?
- [ ] **어절 수**가 해당 레벨의 최소-최대 범위 내인가?
- [ ] 사용된 **연결어미**가 해당 레벨에서 이미 도입된 것인가? (3절 로드맵 참고)
- [ ] 연결어미 **중첩 수**가 해당 레벨의 허용 범위 내인가?
- [ ] L1-L10 어휘를 최대한 활용했는가? (신규 단어 최소화)

### 7.2 스킬 단위 검증

- [ ] 한 스킬 내 문장들의 **복잡도가 고르게 분포**되어 있는가?
- [ ] 한 스킬 내 문장들의 **어절 수 편차가 평균의 40% 이내**인가?
- [ ] 동일 스킬 내 **최소 1개는 해당 레벨의 핵심 티어** 문장인가?

### 7.3 레벨 전환 검증

- [ ] 이전 레벨 마지막 스킬의 문장과 **복잡도 단절이 없는가**?
- [ ] 새로운 티어가 도입될 때 **10-20% 비율로 시작**했는가?
- [ ] 이전 티어가 **즉시 사라지지 않고 점진적으로 감소**했는가?

## 10. 부록: 한국어 문장 복잡도 판별 기준 (Korean Sentence Complexity Criteria)

### 10.1 절(Clause) 수 세기

한국어에서 절의 수는 **서술어(동사/형용사 활용형)의 수**로 판별한다.

```
예시 1: "나는 학생입니다" → 서술어 1개 (입니다) → 1절 = T3
예시 2: "나는 커피를 마시고 빵을 먹습니다" → 서술어 2개 (마시고, 먹습니다) → 2절 = T5
예시 3: "비가 오면 집에서 책을 읽을 것입니다" → 서술어 2개 (오면, 읽을 것입니다) → 2절 = T6
예시 4: "비가 와서 집에 있었는데 친구가 전화해서 나갔습니다" → 서술어 4개 → 4절 = T7
```

### 10.2 T3 vs T4 구별 기준

| 기준 | T3 (단문) | T4 (확장 단문) |
|------|----------|---------------|
| 어절 수 | 2-4어절 | 5-7어절 |
| 부사어 | 0-1개 | 2개 이상 |
| 시간/장소 표현 | 없음 또는 1개 | 1-2개 |
| 예시 | 나는 학생입니다 | 나는 매일 아침 학교에 갑니다 |

### 10.3 T5 vs T6 구별 기준

| 기준 | T5 (중문) | T6 (복문) |
|------|----------|----------|
| 절 관계 | 대등(equal) | 종속(subordinate) |
| 연결 유형 | -고, -지만, 그리고, 하지만 | -서, -니까, -면, -(으)ㄹ 때 |
| 절 독립성 | 각 절이 독립적으로 성립 | 종속절은 독립 불가 |
| 예시 | 나는 커피를 좋아하지만 차도 마십니다 | 비가 오면 집에서 책을 읽습니다 |
