아래 어드민 시스템 수정사항을 반영해주세요.

# 1. 레벨관리

- 레벨관리 기능은 삭제

# 2. 콘텐츠 관리

- 레벨을 선택하면 레벨수정화면 팝업되어야 함.
- 컬럼 헤더는 레벨, 스킬, 문장, 표현이 모두 높이가 동일해야 하며, 위치가 고정되어야 함 (내용만 각 컬럼별로 스크롤되어야 함)
- 항목을 선택하면 우측에 하위 항목 리스트만 되며, 각 항목 내용 오른쪽 끝에는 수정용 아이콘 버튼이 있어서 그것을 클릭하면 수정을 위한 팝업이 표시됨.
- 레벨항목의 레벨 숫자앞에 "Lv"라는 표시붙여서 "Lv1"과 같이 표시되기

# 3. 유형관리 수정

- 항목 선택 시 우측 내용이 조회모드가 아니라 수정모드로 표시되어서 바로 수정가능하도록 함. (단, 생성일시나 상위유형ID 같은 조회용 값은 현재처럼 표시)
- 하위항목이 한단계 들여쓰기 되어서 상위항목과 시각적으로 좀 더 잘 구분되도록 하기
- 생성일은 YYYY-MM-DD hh:mm 형식으로 표시
- 하위항목이 있는 경우 화살표뿐 아니라 항목을 선택해도 하위항목이 펼침과 닫힘이 토글됨

# 4. 기타사항 수정

- 어드민도 favicon 적용 (web에서 적용된 favicon 같이)
- 타이틀 : "Admin"만 표시함
